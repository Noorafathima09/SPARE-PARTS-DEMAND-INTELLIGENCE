
# Spare Parts Demand Intelligence

This project analyzes spare parts sales data from a real-world agency to provide operational demand intelligence. The goal is to support inventory planning, workload management, and branch performance evaluation using Python, SQL, and Power BI.

---

## Project Overview

- **Business Context:** Spare-parts demand is irregular and service-cycle dependent.
- **Objective:** Transform ERP billing data into actionable insights:
  - Identify fast, medium, and slow-moving items
  - Measure demand stability and volatility
  - Track monthly trends for operational planning
  - Support branch and customer-level decisions

## Tools Used
- Python (data cleaning, analysis, visualization)
- SQL (optional)
- Power BI (visualization & dashboard)
